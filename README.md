# 🛡️ AI-Powered Real-Time Cyber Threat Detection Platform

## Overview

The AI-Powered Real-Time Cyber Threat Detection Platform is a Machine Learning-based cybersecurity solution designed to analyze network traffic and identify potential cyber threats in real time. The system processes network activity, evaluates risk levels, and provides threat predictions through an interactive web dashboard.

This project demonstrates the application of Machine Learning, Data Analytics, and Cybersecurity concepts to build an intelligent threat monitoring system.

* Real-Time Network Traffic Analysis
* Machine Learning-Based Threat Detection
* Risk Assessment and Classification
* Interactive Streamlit Dashboard

## Project Workflow

1. Network traffic data is collected and processed.
2. Features are extracted and prepared for analysis.
3. A Machine Learning model is trained on network activity data.
4. The trained model predicts whether incoming traffic is safe or malicious.
5. Results are displayed through an interactive dashboard.
6. Threat analytics and risk assessments are generated for monitoring purposes.

---

## Dashboard Features

### Threat Detection

Users can analyze network traffic by providing:

* Protocol Type
* Packet Size
* Login Attempts

### Risk Assessment

The system calculates a risk score based on traffic behavior and displays a visual risk indicator.

### Threat Analytics

Interactive charts provide insights into network traffic and threat distribution.

### Threat Logs

The dashboard maintains a monitoring table showing recent threat activities and classifications.

cd ai-powered-cyber-threat-detection-platform

### Install Dependencies

pip install -r requirements.txt

### Run Application

python -m streamlit run app.py
